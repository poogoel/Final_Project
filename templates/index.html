A common trope in advanced analytics is that starting with the simplest of means often provides the most success
(think Occam’s Razor theory). In a classification task like this one, logistic regression is the simplest means and the first type of model we used.
.

This is Something

Scikit-learn is a popular Python machine learning library that has many algorithms, including logistic regression. We employed scikit-learn for creating models while using Pandas (a data manipulation library) to pull the files into data frames to be worked on in-memory. Beyond logistic regression, We also could use a common clustering method called k-nearest neighbors along with random forest, Naïve Bayes, and Gaussian process regression.

Also Something

There are several methods to calculate the accuracy of a model so you can compare different options and configurations of those options. One of the most popular is splitting your data into a group to train the models while setting aside a random portion of the data for testing purposes.

Probably Something

For each of the different algorithms we chose, there are settings called hyperparameters. Hyperparameters are like knobs you can turn to tune your model and try to reach greater accuracy. We wrote some quick functions to test the accuracy of each algorithm on the training as compared to the test set as well as plot a graph for each.
